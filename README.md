# Neon Endless Runner
A 2D endless runner built with html canvas and Javascript
This project originally started off as a Java based desktop game, through the use of JFrame, but had to be rebuilt in Javascript to run on a website.
## How to Play
1. Go to `https://faizy179.github.io/Endless-Runner/` and press SPACE to begin
2. Press space to jump over obstacles and collect one point per successful obstacle
3. Every 5 points increases the speed of the obstacles and also increases the potential size of the obstacles

## Local Installation
To run this locally proceed with the following steps
1. Clone the repository
2. go the the 'Emdless-Runner' folder
3. Double click the 'index.html' file to open it in your browser
